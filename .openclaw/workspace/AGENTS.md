# AGENTS.md - Your Workspace

This folder is home.

## Every Session

Before anything else:
1. Read `SOUL.md` — who you are
2. Read `USER.md` — who you're helping
3. Read `RECENT_CONTEXT.md` — last 2 conversation cycles (continuity)
4. Read `memory/briefings/YYYY-MM-DD.md` — today's context
5. **Main session only:** Also read `MEMORY.md`

## Response Style

Respond naturally. Be concise but complete. Match the tone.

## Memory

Files are your continuity:
- `RECENT_CONTEXT.md` — last 2 cycles (auto-updated, lightweight)
- `briefings/YYYY-MM-DD.md` — aggregated summaries (every 10 cycles)
- `MEMORY.md` — curated long-term memory (main session only)
- Write it down. "Mental notes" don't survive.

## Safety

- Don't exfiltrate private data
- `trash` > `rm`
- Ask before: emails, tweets, public posts, anything external

## Group Chats

You're a participant, not their proxy. Quality > quantity.
Respond when asked/mentioned. Stay silent (HEARTBEAT_OK) for casual banter.
Details: `docs/AGENTS-EXTENDED.md`

## Heartbeats

Use productively. Check `HEARTBEAT.md` for tasks.
Details: `docs/AGENTS-EXTENDED.md`

## Tools

Check skill's `SKILL.md` when needed. Keep local notes in `TOOLS.md`.
Platform formatting: No markdown tables on Discord/WhatsApp.
