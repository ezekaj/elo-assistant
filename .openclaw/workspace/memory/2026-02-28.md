# 2026-02-28 - TodoWrite Tool Created

## What
Created session-scoped progress tracking tool based on Claude Code's TodoWrite pattern.

## Files Created
- `/openclaw/src/agents/tools/todo-tool.ts` - Main tool implementation
- `/openclaw/src/agents/tools/todo-tool.test.ts` - 11 passing tests

## Files Modified
- `/openclaw/src/agents/openclaw-tools.ts` - Registered tool with session key

## Schema
```typescript
{
  content: string,      // "Fix the login bug" (imperative)
  status: "pending" | "in_progress" | "completed",
  activeForm: string    // "Fixing the login bug" (present continuous)
}
```

## Behavior
- Stores todos per session (via `agentSessionKey`)
- Returns `oldTodos` + `newTodos` for change tracking
- Auto-clears when all tasks completed
- Helper functions: `getTodosAsPrompt()`, `getTodoReminder()`

## Key Distinction
- OpenClaw's existing `task.ts` = background shell process management
- New `todo-tool.ts` = session-scoped progress tracking (like Claude Code)
